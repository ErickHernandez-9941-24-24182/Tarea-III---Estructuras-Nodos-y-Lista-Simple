# Tarea-III---Estructuras-Nodos-y-Lista-Simple

El programa crea una lista simplemente enlazada para guardar productos usando memoria dinámica.
Define las estructuras Producto y Nodo, crea 4 nodos con new, los enlaza, recorre la lista con un puntero auxiliar para mostrar los datos y sumar los precios, y finalmente libera la memoria con delete para evitar fugas.
